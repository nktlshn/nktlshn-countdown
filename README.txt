# countdown-days
## Countdown to custom date with notifications to Telegram

This is a countdown script, that counts days to DAY X.
It is also sends message with number of days left to your telegram chat, or group, or chanel with a random emoji attached.

You have to have cfg.py file in the repository with constant variables for a project, such as TG_TOKEN, TG_CHAT_ID, DAY_X.

Just a simple project, hello-world to github.
